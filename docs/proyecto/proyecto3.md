# Proyecto Integrador

## **3. Metodología y proceso.**

La documentación deberá evidenciar cómo se desarrolló el trabajo, incluyendo:

investigación;

exploración;

diseño;

iteraciones;

prototipado;

validación;

toma de decisiones.

En proyectos aplicados, el proceso de desarrollo es considerado tan importante como el resultado final.
