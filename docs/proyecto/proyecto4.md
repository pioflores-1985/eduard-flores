# Proyecto Integrador

## **4. Desarrollo tecnico y materialización.**

La documentación deberá describir la resolución técnica del proyecto, incluyendo aspectos como:

prototipo, sistema, dispositivo o software desarrollado;

fabricación y producción;

electrónica, programación y materiales utilizados.

Asimismo, deberá explicarse:

cómo funciona el proyecto;

cómo fue construido;

qué tecnologías utiliza;

por qué se eligieron determinadas soluciones técnicas.

También deberán detallarse:

materiales y componentes utilizados;

partes o sistemas fabricados;

procesos de fabricación implementados:

procesos aditivos,

procesos sustractivos,

otros procesos relevantes;

tecnologías, software y herramientas utilizadas.
