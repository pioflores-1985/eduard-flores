# Proyecto Integrador

## **6. Conclusiones y Proyección.**

La documentación final deberá responder, entre otras, a las siguientes preguntas:

¿Qué aporta el proyecto?

¿Qué conocimientos se comparten?

¿Qué aprendizajes surgieron durante el desarrollo?

¿Qué aspectos podrían continuar desarrollándose?

¿Qué impacto podría tener la propuesta?

¿Qué nuevas preguntas o líneas de trabajo aparecen?

¿Cuáles podrían ser los próximos pasos del proyecto?

¿Qué lo hace innovador o diferente?

¿Cuál es el estado actual del proyecto y el alcance logrado?
